# ChatGPT客户端推荐 ​

## 前言 ​

本文收录了目前市场上最好用的几款ChatGPT同类产品或者替代产品，Windows/Mac/Linux/安卓/苹果等各平台的都有，如果你正在寻找一款好用的ChatGPT客户端，或者已经在使用其中某款，但是觉得不好用，在寻找更好的替代品，这篇文章将会帮助到你。

## 各模型域名填写规则 ​

> **注意**
>
> 以下为常见的域名填写方式，不同客户端可能有所不同，有些可能不需要`/v1`后缀，有些可能需要填写完整的`/v1/chat/completions`后缀，具体请查阅对应客户端的官方文档。

| 模型供应商 | 域名                                    | 备注           |
| :--------- | :-------------------------------------- | :------------- |
| OpenAI     | `https://api.openai-proxy.org/v1`     | 一般需要加`/v1` |
| Anthropic  | `https://api.openai-proxy.org/v1`   |                |
| Gemini     | `https://api.openai-proxy.org/v1`   |                |

## 注意事项 ​

1.  **小白用户经常会问的一个问题：** 为什么通过提问的方式，问GPT自己是不是GPT4，他说自己是GPT3? 这个问题是刚接触API的小白最经常有的困惑，已经被写到文档里了[《常见问题》](https://123)
2.  如果报错 `404 Not Found`，一定是域名配置的不对，请仔细阅读对应客户端的使用文档。
3.  很多人会误把Claude等API域名配置到OpenAI的地方，导致配置错误无法使用，一定要注意。

## Chatbox ​

ChatBox是类似ChatGPT的一款本地桌面版客户端，支持Windows、Mac和Linux。下载安装即可使用。完全兼容GPT API Proxy的API，支持ChatGPT、Claude 3、Gemini等多种模型。缺点是不支持插件调用，PDF阅读等高级功能需要购买单独的月费会员，会员费价格和ChatGPT差不多，如果需要网络搜索、文件阅读之类的能力的话，需要考虑其他客户端，目前在这方面Chatbox还无法替代ChatGPT。

| 功能                     | 是否支持 |
|------------------------|----------|
| 图片识别与理解           | 支持     |
| Dalle图片生成            | 支持     |
| SD图片生成               | 不支持   |
| 插件支持                 | 不支持   |
| 记忆能力                 | 不支持   |

站长评价：🌟🌟🌟🌟

## 效能类客户端推荐

提示

效能类虽然支持聊天，但是一般不作为主力聊天工具，而是通过聊天辅助来提高工作效率。

## Cursor

市面上最强的AI开发编辑器，目前效果最好，支持composer多文件同时编辑，暂时不支持中文。具体使用网上有很多Cursor使用教程和技巧，这里不再赘述。

如何付费：

1. 可以充值购买Cursor的会员订阅，付款需要境外卡，国内信用卡不行，网上也能找到一些会员合租的。
2. Cursor订阅每月有使用额度限制，也可以配置GPT API Proxy的API Key使用，无额度限制。

在设置页面配置好API Base和API Key即可，注意Cursor虽然支持自定义Custom Claude/Gemini的API Key，但是不支持自定义API Base，因此无法在Cursor内使用我们的Claude/Gemini。

![](https://public-img-bucket.oss-cn-beijing.aliyuncs.com/openai-asia/202411302301884.png)

### 地址

## Zed

Zed也是一款AI编辑器，由Atom编辑器原创团队新作，支持所有GPT API Proxy模型。

在Zed的配置项里添加如下配置，即可接入所有GPT API Proxy的模型：

```json
{
    "language_models": {
        "openai": {
            "version": "1",
            "api_url": "https://api.openai-proxy.org/v1"
        },
        "anthropic": {
            "version": "1",
            "api_url": "https://api.openai-proxy.org/anthropic"
        },
        "google": {
            "api_url": "https://api.openai-proxy.org/google"
        }
    }
}
```

## 沉浸式翻译

沉浸式翻译是一款优秀的浏览器翻译插件，支持OpenAI作为翻译模型，可以购买会员，也可以搭配GPT API Proxy无限额度使用，由于翻译工具普遍对并发要求较高，因此搭配使用支持高并发的GPT API Proxy效果更好。

注意沉浸式翻译必须使用完整的路径后缀：/v1/chat/completions。

![](https://public-img-bucket.oss-cn-beijing.aliyuncs.com/openai-asia/202411302308060.png)

## OpenAI Translator

一款本地翻译软件，和沉浸式翻译有重叠的部分，但是这个支持桌面端翻译。

在设置页面配置我们的域名，域名和路径是分开的。

![](https://public-img-bucket.oss-cn-beijing.aliyuncs.com/openai-asia/202411302316218.png)

## GPT Academic

一个使用GPT进行学术优化的工具，一般用于科研用途的论文翻译等，针对论文翻译领域有一定的优势，支持GPT API Proxy的API Key，具体配置参考GPT Academic使用教程。
