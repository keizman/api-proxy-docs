# 平台简介 ​

## 我们的故事 

凭借团队多年一线大厂积累的研发经验，我们打造了:
- 业界服务质量最高的OpenAI自研中转平台
- 扩展到Anthropic(Claude) API、Gemini API等其他AI平台
- 依靠100%官转的信誉保障赢得头部企业客户与顶级高校科研机构认可

## 服务定价 ​

GPT API Proxy是实打实付费的官方账号转发（也叫官转），都是正经渠道长期充值养出来的账号，没有黑产那些套路，因此会在官方价格基础之上加收一定比例服务费用，这些费用大部分用于覆盖Stripe等支付渠道费、汇兑费用、带宽费用、服务器费用、封号成本、日常运营费用等，具体定价可以参考模型定价页面

主打一个靠谱稳定不折腾，节省时间成本，为各类企业的商业需求和高校的科研机构提供一个长期稳定靠谱的官方API转发，我们相信如果您的业务真的很重要，这部分额外成本是值得的。



## 是否支持ChatGPT ​

支持。部分非开发者用户可能会发现我们平台提供的服务和您想象中的不太一样，这是因为ChatGPT和API其实完全是2个产品，不能混淆在一起，ChatGPT是面向普通大众的Web聊天服务，而API是面向专业开发者提供的用来构建AI产品的接口服务，需要一定的代码开发能力才能使用。
