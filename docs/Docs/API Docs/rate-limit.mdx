## 限流提升

企业用户可能需要大规模调用平台接口，为了保证每个用户得到的资源能够合理分配，目前需要人工进行审核，请完成开发者认证后，邮件联系客服沟通申请，我们会根据您的需求进行评估。

### 申请提升所需信息

邮件内容请至少包括以下内容：

| 序号 | 申请必填信息 |
|------|------------|
| 1 | 计划要高并发使用的模型 |
| 2 | 每个模型的预计TPM和RPM |
| 3 | 描述下具体的使用场景，为什么需要高并发使用，比如是用于数据处理还是AI产品开发等 |
| 4 | 计划使用的时长，是短期需求，还是长期需求 |
| 5 | 预估消费的金额规模 |
| 6 | 要申请的账号邮箱 |
