# 平台简介 ​

## 国内第一家API中转平台 ​

[GPT API Proxy](https://www.GPT API Proxy-asia.com) 是国内最早的OpenAI代理平台，原名OpenAI Asia，目前拥有包括阿里、腾讯、百度在内的各类大型企业客户，以及清华大学、北京大学等数十所国内高校科研机构客户，是亚洲规模最大的商用级的OpenAI代理转发平台，定位于企业级商用需求，面向企业客户的线上服务和国内科研机构提供高质量稳定的官方OpenAI API、Anthropic(Claude) API、Gemini API等AI平台接口中转服务，在专业开发者群体里享有非常高的知名度和口碑。

创始团队均来自国内一线大厂核心骨干，拥有世界500强级别的商用服务研发经验，熟悉一线大厂研发管理流程。团队负责人 [无为](https://x.com/noheroishere) 曾任头部大厂云计算领域技术负责人，领导过技术中台多项核心产品的建设工作，作为首席架构师参与了GPT API Proxy整个平台的全部架构设计。

## 我们的故事 ​

2022年底ChatGPT火爆全球，GPT API Proxy团队凭借500强企业的内部关系，第一时间获得了GPT4 API的开发者权限（国内第一个GPT-4-32k权限），用于我们内部的AI产品测试。彼时付费充值购买ChatGPT API是一个很大的门槛，需要境外银行卡绑卡、境外手机号等，我们当时写过很多教程，如何购买和申请OpenAI API，但OpenAI审核严格，很多人还是很难获得到权限。当看到很多开发者为了获取GPT4 API Key一筹莫展时，我们决定开发一个平台来分享GPT4权限给其他开发者使用，于是迅速推出了OpenAI Asia，成为了业界第一家提供OpenAI API代理服务的专业开发者平台，在V2EX（知名开发者论坛）上一经发布便收货了大量种子用户。

随后凭借着团队多年一线大厂积累的研发经验，打造了业界服务质量最高的OpenAI自研中转平台，业务范围也扩展到了Anthropic(Claude) API、Gemini API等其他AI平台，并依靠100%官转的信誉保障，陆续收获了大量头部企业客户青睐，也获得了众多顶级高校科研机构的认可，成为了业内规模最大的OpenAI商用级第三方代理转发平台。

我们坚信未来AI服务会成为和水电一样的基础服务，同时我们团队自身也在依赖GPT API Proxy作为底层基石开发各式各样的AI领域应用，与所有开发者共建未来的AI生态。GPT API Proxy始于开发者，定位于开发者，会以顶级云服务的标准要求自身，为所有专业开发者提供好基石服务。

## 适合哪些用户 ​

1. 有商业级生产服务或者科研学术用途，需要长期稳定靠谱的OpenAI/Claude/Gemini官方API转发服务，同时有较高的服务可用性，不用担心线上业务稳定性。
2. 有大规模数据处理需求，需要大量Tier5级账号才能处理的高TPM并发需求。

4. 正常使用但是却被OpenAI、Claude频繁封号过的用户，不想再折腾。

## 是否支持ChatGPT ​

支持。部分非开发者用户可能会发现我们平台提供的服务和您想象中的不太一样，这是因为ChatGPT和API其实完全是2个产品，不能混淆在一起，ChatGPT是面向普通大众的Web聊天服务，而API是面向专业开发者提供的用来构建AI产品的接口服务，需要一定的代码开发能力才能使用。

对于只想聊天使用的客户，GPT API Proxy拥有纯自研的ChatGPT替代品：CloseChat。一比一复刻了ChatGPT的界面和功能，支持网页访问和Windows/Mac/Linux多平台客户端，是目前市面上功能最强，用户体验最好的客户端。直接使用GPT API Proxy账号登录<https://www.closechat.org>即可，无需任何配置，和普通客户端使用API KEY效果一样，按量付费，没有任何月费。

![CloseChat](https://www.closechat.org/gen-image.jpg)

CloseChat可以完美替代比如Chatbox、Botgem、NextChat、LobeChat、TypingMind等三方客户端，但是如果您确实需要使用这些客户端，可以参见教程[《ChatGPT客户端推荐》](/tutorial/integrations/chatgpt-client.html)进行配置。
